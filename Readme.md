# Todo Task Card

A sleek, responsive Task Card component designed for modern web applications. 

## Features
- **Interactive Edit Mode**: Easily edit task details including title, description, priority, and due date within a hidden panel.
- **Dynamic Status Tracking**: Seamlessly switch between states (Pending, In Progress, Done) which are automatically synced across checkboxes and dropdowns.
- **Priority Indicators**: Visual cues (color-coded bars and badges) for indicating the task's structural urgency level.
- **Granular Due Dates & Overdue Flags**: Calculates the exact time remaining, dynamically highlighting tasks that are due soon or instantly flagging overdue constraints.
- **Collapsible Layout**: Keeps the UI tidy by easily collapsing overly long descriptions behind an interactive expandable toggle.

Built beautifully with pure HTML, CSS, and vanilla JavaScript.
